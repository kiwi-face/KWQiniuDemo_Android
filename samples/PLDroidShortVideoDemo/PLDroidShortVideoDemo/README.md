# trackerAndroidDemo

![kiwi-logo](kiwi-logo.png)

##trackerAndroidDemo是什么?

基于七牛直播SDK v2.1.1实现直播中多人脸track／多人脸动态贴纸的Android demo

##trackerAndroidDemo有哪些功能？

* 使用kiwi的`人脸跟踪库`，实现多人脸跟踪
* 在七牛直播中，根据人脸跟踪结果实现`人脸动态贴纸`
* 在七牛直播中，实现`哈哈镜`，`瘦脸`功能（即将提供）
