# PLDroidCameraStreaming Release Notes for 1.3.8

## 简介
PLDroidCameraStreaming 是为 Android 开发者提供的 RTMP 直播推流 SDK

## 记录

### 推流 SDK
  - 发布 pldroid-camera-streaming-1.3.8.jar
  - 更新 libpldroid_streaming_core.so
  - 优化切换前后置摄像头数据重发时间，增强推流过程中切换前后置摄像头的稳定性
  - 优化内存使用，避免 OOM
  - 修复部分机型概率性 crash 问题
  - 兼容 supportedPreviewSizeList 为空的机型
